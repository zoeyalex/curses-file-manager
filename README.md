# curses file manager
a terminal based file manager with vim-like keybindings

Navigation
-----
* `q`: quit
* `o`: cd inside a directory or open a text file on the adjacent panel
* `p`: cd ..
* `h`: switch panel left
* `l`: switch panel right
* `k`: move up
* `j`: move down
